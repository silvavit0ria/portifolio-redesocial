Sacolão Viva

Este projeto foi desenvolvido com o objetivo de praticar JavaScript. Ele permite que o usuário busque um item desejado e verifique seu valor. Caso o item não esteja disponível, uma mensagem será exibida. O sistema aceita apenas letras minúsculas e impede o usuário de clicar no botão "pesquisar" se o campo estiver vazio
